
React-16.x application with header and footer using React-router-4 and Redux-form.

=================================


### Import the project from github

  ```sh
  git clone https://github.com/dspoonia7/react-spa.git
  ```


### Install dependencies

  ```sh
  npm install
  ```


### Start build with

  ```sh
  npm start
  ```


### Navigate to 

  http://localhost:3000
